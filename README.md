## ğŸš€ Getting Started

### Prerequisites
- Flutter SDK `^3.24.0`
- Dart SDK `^3.5.0`

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/leet-scroll.git
   cd leet-scroll
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run Code Generation:**
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```

4. **Run the app:**
   ```bash
   flutter run
   ```
